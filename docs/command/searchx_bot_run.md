## searchx bot run

Run the bot

```
searchx bot run [flags]
```

### Examples

```
searchx bot run -c config/bot/config.min.yaml
```

### Options

```
  -h, --help   help for run
```

### Options inherited from parent commands

```
  -c, --config string   the path to the config file (default "config/bot/config.min.yaml")
```
